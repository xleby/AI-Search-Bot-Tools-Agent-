# 🤖 AI Search Bot — Tools Agent

Telegram-бот с поиском в интернете, работающий через function calling (tools).
Модель сама решает когда искать, что искать, и как ответить.

## Возможности

- 🔍 **Поиск в интернете** — модель сама составляет запросы на русском и английском
- 🧠 **Контекст разговора** — кнопка «🔍 Уточнить» для продолжения темы
- 📝 **Качественные ответы** — парсинг статей с фильтрацией по источникам
- 🔄 **Авто-реконнект** — восстанавливается при обрыве связи

## Установка

```bash
# 1. Копируем .env.example в .env
copy .env.example .env

# 2. Заполняем .env (см. ниже)

# 3. Устанавливаем зависимости
pip install -r requirements.txt

# 4. Запускаем
python main.py
```

## Настройка .env

```env
# Токен бота — получить у @BotFather
BOT_TOKEN=123456789:ABCdef...

# HTTP прокси для обхода блокировок Telegram API
# Cloudflare Workers reverse proxy (см. инструкцию ниже)
HTTP_PROXY=https://your-proxy.workers.dev

# Локальный SearXNG
SEARXNG_URL=http://127.0.0.1:8888

# Qwen LLM сервер (OpenAI-compatible API)
MODEL_URL=http://127.0.0.1:8080
```

## Требования

| Компонент | Описание |
|-----------|----------|
| **SearXNG** | Метапоисковик, запущенный локально (`http://127.0.0.1:8888`) |
| **Qwen LLM** | Локальная модель с поддержкой tools/function calling |
| **HTTP прокси** | Cloudflare Workers reverse proxy (для обхода блокировок) |

### Настройка HTTP прокси (Cloudflare Workers)

1. Создайте Worker на [dash.cloudflare.com](https://dash.cloudflare.com)
2. Вставьте код reverse proxy для Telegram API (см. документацию вашего провайдера)
3. Укажите URL в `.env`: `HTTP_PROXY=https://your-worker.workers.dev`

## Архитектура

```
Пользователь → "Что такое TurboQuant?"
    ↓
[aiogram] получает сообщение
    ↓
[LLMAgent] отправляет вопрос модели + описание tool "search_web"
    ↓
[Qwen модель] решает: "Нужно поискать" → вызывает tool
    ↓
[SearchTool] ищет в SearXNG → парсит лучшие статьи → возвращает текст
    ↓
[Qwen модель] анализирует результаты → генерирует ответ
    ↓
[aiogram] отправляет ответ + кнопку «🔍 Уточнить»
```

## Фильтрация источников

Бот автоматически фильтрует результаты поиска:

1. **Блок-лист** — спам, сокращатели URL, мусорные агрегаторы
2. **Оценка сниппета** — короткие/пустые описания идут в конец
3. **Приоритет доменов** — Wikipedia > Habr > СМИ > Reddit > блоги
4. **Адаптивный парсинг** — сначала 3 лучших, если мало текста — ещё 2

## Как работает кнопка «Уточнить»

- Обычные вопросы → **без истории** (новая тема)
- Нажата «🔍 Уточнить» → **следующий вопрос с историей** разговора
- После одного уточняющего вопроса → история отключается
